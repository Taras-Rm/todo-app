# ToDo App (client)

This is a basic todo application.

## Technologies

- Next.js
- Axios

## Run app

1. Clone repository:
``` bash
git clone https://github.com/Taras-Rm/todo-app.git
```

2. Move to client folder:
``` bash
cd ./client
```

3. Install dependencies
``` bash
npm install
```

4. Create .anv file according to .env.example

5. Start app
``` bash
npm run dev
```
